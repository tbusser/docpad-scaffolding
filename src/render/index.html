Welcome

Welcome to the Docpad scaffolding project.

Partials

There a couple of default partials included.

head

This partial will render the head element of the page. This partial uses the following meta data for configuration:

document.title
In the YAML of the page you can specify the title property of the page.
document.isInvisble
When this property is specified for the page and it's value is 1, the following meta tag will be rendered in the head of the page: <meta name="robots" content="noindex, follow">
noIndex
You can also hide the page from search enginges by passing the variable noIndex=1 along in the partial parameters. This can be done by calling the partial in the following way: {{ partial 'document/head' noIndex="1" }}
